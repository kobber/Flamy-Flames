# Flamy Flames

A small Minecraft mod that improves the look of flame particles by removing the scaling animation, changing their rotation mode and adding animated textures based on age.

Affects all vanilla and modded blocks that use flame particles, like torches, candles and lit furnaces.

## Does this work with resource packs?

The animated textures can be overriden by resource packs. This will probably look fine with packs that add their own animated textures, but will look less than ideal with packs that just change the vanilla flame texture.
Because of this, the textures of this mod are provided as a built-in resource pack to allow reordering if needed.

## Mod Compatibility

Animated textures are included for the flame types added by:
* **Buzzier Bees** (Soul Candle flame)
* **Caverns and Chasms** (Cupric flame)
* **Copper Age Backport** (Copper flame)

New flame particles added by other mods will still get the the rotation and removed scaling, but won't have animated textures.
